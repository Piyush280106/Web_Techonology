
 CONCEPTS OF WEB TECHNOLOGY


What is Web Technology?
Web Technology means all the tools and languages that are used to create websites and web applications.
 It helps us design, build, and run websites on the internet.
 Examples of web technologies are HTML, CSS, JavaScript, servers, and browsers.

 Real-time examples of Web Technology
 1. Google Search – when we search something, web technology shows results instantly.
 2. Instagram – uploading photos, liking posts, and chatting are done using web technologies.
 3. Online Shopping Websites – adding items to cart and making payments use web technology.
 4. Online Classes – video calls, chat, and screen sharing are possible because of web technology.

 What is JavaScript?
 JavaScript is a programming language used to make websites interactive.
 It helps us create things like buttons, pop-ups, form validation, and animations.
 Without JavaScript, websites would be static and boring.

 Which file extension is used for JavaScript?
 JavaScript files are saved using the .js extension.
 Example: script.js, app.js, concepts.js

 Why do we use Node.js?
 Node.js allows us to run JavaScript outside the browser.
 It helps us build backend applications like servers and APIs.
 Using Node.js, JavaScript can be used for both frontend and backend development.
